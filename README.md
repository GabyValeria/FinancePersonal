# 💰 Finanças Pessoais (Em Desenvolvimento)

🚧 **Este projeto ainda está em desenvolvimento. Funcionalidades podem mudar e partes do sistema ainda estão sendo implementadas.** 🚧

---

## 📌 Descrição

Sistema web de controle de finanças pessoais, desenvolvido com **Flask**, **HTML5**, **CSS3** e **JavaScript puro**.  
Permite o acompanhamento de receitas, despesas, planejamento financeiro e visualização de dados no painel (dashboard).

## 🎯 Funcionalidades (previstas)

- Dashboard com resumo financeiro
- Cadastro e listagem de transações (receitas e despesas)
- Edição e remoção de transações
- Gráficos interativos com JavaScript
- Planejamento financeiro mensal
- Filtro por data/categoria
- Responsividade total para mobile
- Exportação de dados (CSV/JSON)

## 🛠️ Tecnologias Utilizadas

- Backend: [Flask](https://flask.palletsprojects.com/)
- Frontend: HTML5, CSS3, JavaScript
- Banco de dados: SQLite (ou JSON local)
- Gráficos: JavaScript (Canvas ou bibliotecas puras)
- Estilo: CSS puro com design limpo e responsivo
